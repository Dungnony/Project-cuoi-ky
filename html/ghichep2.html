📝 Ghi Chép Học Tập: Làm Quen Với HTML, CSS và JavaScript

✨ "Học web không chỉ là viết code, mà còn là hành trình xây nên thế giới số của riêng bạn."

Đăng bởi Dũng vào ngày 8 tháng 5 năm 2025

HTML (HyperText Markup Language) là ngôn ngữ dùng để tạo cấu trúc trang web.


<!DOCTYPE html>
<html>
<head>
  <title>Trang web đầu tiên</title>
</head>
<body>
  <h1>Xin chào!</h1>
  <p>Chào mừng bạn đến với blog học lập trình.</p>
</body>
</html>

🧠 Ghi nhớ:

h1 đến h6: tiêu đề từ lớn đến nhỏ

p: đoạn văn bản

img, a, ul, div... là các thẻ cấu trúc khác

🔹 Phần 2: CSS – Trang điểm cho trang web

CSS (Cascading Style Sheets) dùng để thiết kế giao diện: màu sắc, font chữ, bố cục,...

📌 Ví dụ: Thêm màu và căn giữa tiêu đề


<style>
  h1 {
    color: #2c7a7b;
    text-align: center;
    font-family: 'Segoe UI', sans-serif;
  }

  p {
    font-size: 18px;
    line-height: 1.6;
  }
</style>

📎 CSS có thể viết ở:

Trong thẻ style (nội tuyến)

Tệp .css riêng: style.css

Gắn vào thẻ (inline): h1 style="color: red">/h1

🔹 Phần 3: JavaScript – Tạo sự tương tác

JavaScript giúp trang web có hành động, ví dụ: khi click, hiển thị cảnh báo, xử lý form,...

📌 Ví dụ: Khi nhấn nút sẽ hiện alert


<button onclick="sayHello()">Nhấn tôi!</button>

<script>
  function sayHello() {
    alert("Xin chào bạn!");
  }
</script>

🧠 JS có thể làm được:

Bắt sự kiện (click, nhập liệu,...)

Thay đổi nội dung HTML

Kiểm tra dữ liệu form

Giao tiếp với server (AJAX)

💬 Kết luận

Học HTML, CSS, JS là bước đầu tiên để trở thành lập trình viên web. Dù mới bắt đầu, nhưng mỗi dòng code là một bước tiến. Cứ kiên trì, bạn sẽ thấy thành quả!
